Gemini API Key
ขอ key ฟรีได้ที่ไหน?
  1. เปิด aistudio.google.com → ล็อกอิน Google
  2. กด "Get API key" → "Create API key"
  3. คัดลอกมาวางที่นี่
3
Google Client ID (OAuth)
วิธีสร้าง Client ID
  1. เปิด console.cloud.google.com → สร้าง Project
  2. เปิดใช้ Google Sheets API และ Google Drive API (APIs & Services → Library)
  3. APIs & Services → Credentials → Create Credentials → OAuth client ID → Web application
  4. เพิ่ม Authorized JavaScript origins = URL ที่เปิดแอปนี้ (เช่น http://localhost:8899)
  5. คัดลอก Client ID มาวาง
⚠️ ตอนนี้เปิดไฟล์แบบ file:// — Google OAuth ต้องเปิดผ่าน http(s) เช่น localhost หรือโฮสต์จริง จึงจะล็อกอินได้ (ยังตั้งค่าเก็บไว้ก่อนได้)
